# LUMA Explorer

Reference viewer for inspecting LUMA artifacts.

## Inputs
- `scene_ir.json` – canonical semantic structure
- `render.svg` – static plate
- `animation_plan.json` – motion instructions

## Features
- Pan / zoom (mouse drag + wheel)
- Hover inspection (entity JSON)
- Time scrub (drives animation plan)
- Toggle pulse / decay / transfer layers

## Usage
Open `index.html` in a browser (no build step).

This viewer is **read-only** and must not be used as a source of truth.
